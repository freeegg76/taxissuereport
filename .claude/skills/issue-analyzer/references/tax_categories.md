# 세목 분류 기준

국세 및 지방세의 표준 분류 기준입니다. 이슈 분석 시 아래 세목 중 가장 관련성이 높은 것을 선택하세요.

## 국세 (National Tax)

| 세목 | 주요 관련 법령 | 대표 쟁점 |
|------|--------------|---------|
| 소득세 | 소득세법 | 근로소득, 사업소득, 양도소득, 종합소득 |
| 법인세 | 법인세법 | 손금불산입, 부당행위계산부인, 세무조정 |
| 부가가치세 | 부가가치세법 | 과세·면세 판단, 매입세액공제, 공급 시기 |
| 상속세 | 상속세 및 증여세법 | 과세가액, 공제, 평가 |
| 증여세 | 상속세 및 증여세법 | 증여재산가액, 증여시기, 완전포괄주의 |
| 종합부동산세 | 종합부동산세법 | 주택수 판정, 합산배제, 공정시장가액비율 |
| 개별소비세 | 개별소비세법 | 과세 물품, 과세 시점 |
| 교육세 | 교육세법 | 금융업·보험업 등 |
| 농어촌특별세 | 농어촌특별세법 | 감면세액의 부가세 성격 |
| 주세 | 주세법 | 주류 분류, 세율 적용 |

## 세목 판단 우선순위

이슈 텍스트에서 다음 단서를 찾아 세목을 판단하세요:

1. **법령 명시**: "소득세법", "법인세법" 등이 직접 언급된 경우 해당 세목 선택
2. **거래 유형**: "부동산 양도" → 양도소득세(소득세), "법인 결산" → 법인세
3. **납세자 유형**: "개인 사업자" → 소득세 또는 부가가치세, "법인" → 법인세
4. **키워드**: "매입세액", "세금계산서" → 부가가치세
