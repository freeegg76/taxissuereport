# 국가법령정보 판례 API 명세 요약

## 기본 정보

- **기관**: 법제처 국가법령정보센터
- **공식 문서**: https://www.law.go.kr/LSW/openApiInfo.do
- **기본 URL**: `https://www.law.go.kr/DRF`
- **인증**: OC (Open API Key) 파라미터로 전달

## 판례 목록 조회 API

**엔드포인트**: `GET /lawSearch.do`

| 파라미터 | 필수 | 설명 | 예시 |
|---------|------|------|------|
| OC | Y | API 인증키 | your_api_key |
| target | Y | 검색 대상 | `prec` (판례) |
| query | Y | 검색어 | `부당행위계산부인` |
| type | Y | 응답 형식 | `JSON` |
| display | N | 검색 결과 수 | `20` (기본값) |
| page | N | 페이지 번호 | `1` |

**응답 구조**:
```json
{
  "PrecSearch": {
    "totalCnt": "100",
    "prec": [
      {
        "판례일련번호": "123456",
        "사건명": "법인세부과처분취소",
        "사건번호": "대법원 2019두38656",
        "선고일자": "20200101",
        "법원명": "대법원",
        "사건종류명": "세무",
        "판결유형": "판결",
        "판시사항": "...",
        "판결요지": "...",
        "참조조문": "법인세법 제52조",
        "참조판례": "...",
        "판례내용": "전문 URL"
      }
    ]
  }
}
```

## 판례 본문 조회 API

**엔드포인트**: `GET /lawService.do`

| 파라미터 | 필수 | 설명 |
|---------|------|------|
| OC | Y | API 인증키 |
| target | Y | `prec` |
| ID | Y | 판례일련번호 |
| type | Y | `JSON` |

**응답 구조**:
```json
{
  "PrecService": {
    "판례일련번호": "123456",
    "사건명": "...",
    "판례내용": "HTML 형태의 전문"
  }
}
```

## 주요 검색 전략

### 세목별 추천 검색어

| 세목 | 추천 검색어 |
|------|-----------|
| 법인세 | `부당행위계산부인`, `손금불산입`, `세무조정` |
| 소득세 | `양도소득세`, `사업소득`, `실질과세` |
| 부가가치세 | `매입세액공제`, `세금계산서`, `면세` |
| 상속·증여세 | `증여재산`, `상속공제`, `평가액` |

### API 호출 제한
- 일일 최대 호출: 1,000회
- 한 번에 최대 조회: 20건
- 재시도: 오류 시 5초 후 최대 3회
